# 키캡 키링 (Keycap Keyring) 🐰⌨️

아이들이 가지고 노는 **키캡 키링 장난감**을 웹앱으로 재현한 프로젝트입니다.
투명 키캡 안에 좋아하는 도안을 넣고, 기계식 키보드처럼 딸깍딸깍 눌러보세요!

> 참고 영상: [무료 도안 30종 — 취향대로 꾸미는 키캡 키링 만들기](https://youtu.be/X7hP01I5bi0)

## 기능

- **실물 느낌의 2.5D 키캡** — 투명 키캡·광택·스위치 스템·카라비너 키링까지 SVG로 재현, 손가락을 따라 눌리고 튀어오릅니다
- **리얼 기계식 스위치 사운드** — 청축(찰칵)·갈축(톡톡)·적축(부드러움) 3종을 Web Audio로 실시간 합성 (음원 파일 없음, 매번 미세하게 다른 소리)
- **도안 꾸미기** — 이모지 24종 + 패턴 6종 프리셋, 이름 2글자 직접 새기기
- **클릭 카운터** — 딸깍 횟수 저장 (localStorage)
- **LED 불빛** — 누를 때 키캡 속이 은은하게 빛남, 색상 선택 가능
- **햅틱 진동** — 지원 기기(Android Chrome 등)에서 타건감 전달
- **오프라인 지원** — 한 번 열면 인터넷 없이도 동작 (PWA)

## 실행 방법

빌드 도구가 필요 없는 정적 사이트입니다.

```bash
python3 -m http.server 8080
# http://localhost:8080 을 폰/브라우저에서 열기
```

GitHub Pages 등 정적 호스팅에 그대로 배포할 수 있습니다.

## 알아두기

- iOS(아이폰)는 **무음 모드에서 소리가 나지 않습니다** — 무음 스위치를 꺼 주세요.
- iOS Safari는 웹 진동 API를 지원하지 않아 진동 옵션이 숨겨집니다.

## 구조

```
index.html            # 단일 페이지 + 키캡 SVG 장면
css/                  # base(뷰포트 잠금) / keycap(장면·애니메이션) / ui(설정 시트)
js/main.js            # 엔트리 — 상태·입력·사운드·저장 연결
js/keycap.js          # 누름 상태 머신 (Pointer Events + 멀티터치)
js/audio/engine.js    # AudioContext 수명주기, 마스터 버스+컴프레서
js/audio/switches.js  # 청/갈/적축 사운드 합성 레시피
js/designs.js         # 도안·색상 프리셋 데이터
js/settings.js        # 설정 바텀 시트
js/storage.js         # localStorage 영속화
sw.js                 # 오프라인 캐시
```
